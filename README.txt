HackHive Guide 26 Website

HOW TO RUN:
1. Make sure you have Node.js installed on your computer.
2. Open this unzipped folder in VS Code.
3. Open a new Terminal in VS Code (Terminal -> New Terminal).
4. Run this command to install the necessary files:
   npm install

5. After the installation is complete, run this command to start the website:
   npm run dev

6. Click the link that appears in the terminal (usually http://localhost:5173/) to view the website.
